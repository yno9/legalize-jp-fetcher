# legalize-jp

Japanese legislation as Markdown, version-controlled with Git.

One file per law. One commit per amendment.

**Source:** [e-Gov Law API v2](https://laws.e-gov.go.jp/api/2) — Ministry of Internal Affairs and Communications

Seeking to join the [Legalize](https://legalize.dev) project.

## Quick Start

```bash
git clone https://github.com/yno9/legalize-jp.git
git log --oneline -- jp/129AC0000000089.md  # amendment history of 民法
```

## Structure

```
jp/
  321CONSTITUTION.md    — Constitution of Japan
  129AC0000000089.md    — Civil Code (民法)
  140AC0000000045.md    — Penal Code (刑法)
  {law_id}.md           — flat structure, law_id as filename
```

## Frontmatter

Each file has YAML frontmatter with `title`, `identifier`, `country`, `rank`, `publication_date`, `last_updated`, `status`, `source`, and `extra` fields.

Commit types: `[bootstrap]` (initial version), `[reforma]` (amendment).

## Notes

Amendment history is available from approximately 2016 onwards due to e-Gov API limitations. Earlier revisions are not available.
